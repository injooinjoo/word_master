# Vocab 오답( distractors ) 검토 요약

## 1. 오답 채우는 원칙 (시험 출제자 관점)

**문제 형태:** "다음 단어의 뜻으로 가장 적절한 것은?" (4지선다)

- **오답 = 정답이 아니지만, 시험에서 실제로 나올 법한 선택지**
  - 같은 **품사**로 통일 (명사 뜻 → 명사 뜻, 동사 뜻 → 동사 뜻).
  - **헷갈리기 쉬운** 다른 단어의 뜻: 비슷한 문맥·주제, 발음/철자 유사 단어의 뜻.
- **오답 = "비슷한 것 중에서 좀 다른 것"** (아무 단어나 나열 금지)
  - **(1) 영어 발음/철자 비슷한 단어의 다른 뜻** — event(행사), ever↔every(모든), exit↔exist(존재하다), fair↔fare(요금) …
  - **(2) 반대되는 단어(반의어)** — 2~3개.
  - **(3) 한글 발음이 비슷한 단어** — 헷갈릴 수 있는 다른 뜻.
  - **(4) 같은 분야·유사 개념이지만 다른 단어의 뜻** — 예: 시험↔숙제·퀴즈·과제, 경험↔실험·이론·추측.
  - 위 네 가지를 **섞어서** 오답 10개 구성. 오답들끼리도 한 유형만 나열하지 말 것.
- **금지**
  - **공통 리스트 반복** (의사, 연필, 냉장고, 책, 산, 달리다 … 같은 걸 여러 단어에 그대로 재사용).
  - 정답과 **동의어·유사어** (예: 영화관↔극장, 새벽↔황혼).
  - 정답과 **같은 분류만** 나열 (예: 새벽 문제에 해질녘·한낮·저녁만).
  - **풀 형태** 나열: `정보, 사물, 행동, 사건, 상태, 개념…`.
  - **품사 불일치**: 명사인데 동사 뜻만 오답으로 넣기 (또는 그 반대).

## 2. 현재 상태 (풀 형태 오답 잔여)

| 구간        | 풀 형태/일반 오답 잔여 |
|------------|------------------------|
| 배치 01~22 | 소수 (1~4건)           |
| 배치 23~32 | 소수 (1~2건)           |
| 배치 33~39 | 다수 (27~43건)         |
| 배치 40~70 | 다수 (26~40건)         |

- **배치 01**: 오답이 과일/동물/동작 등 **같은 범주 다른 항목**으로 잘 채워져 있음 (기준 참고).
- **배치 40, 66** 등: 여전히 `정보·사물·행동…` 또는 `시작하다·중단하다·선택하다…` 같은 **공통 리스트**가 많음.

## 3. 오답 예시 (시험 스타일)

| 단어        | 정답     | 품사   | 시험 스타일 오답 예시 |
|------------|----------|--------|------------------------|
| cinema     | 영화관   | noun   | (1)시네마→유사:극장 (금지) 대신 (2)반의:집, 밖 (3)유사개념:공연, 상영, 영화제, 스크린, 좌석, 관람 (4)발음유사:scene(장면), see(보다) 뜻 |

## 4. 권장 작업

1. **풀 형태 오답 전부 제거**  
   `정보, 사물, 행동…` / `시작하다, 중단하다…` 등 → 각 단어 **뜻·품사**에 맞는 시험 스타일 오답으로 교체.

2. **동의어·유사어 점검**  
   정답과 뜻이 겹치거나 헷갈리는 오답(극장, 해질녘, 쇼파, 읍·면·동 등) 제거 후 다른 범주/의미로 교체.

3. **품사 일치**  
   명사 항목에 동사 뜻만 오답으로 들어간 경우 → 명사 뜻으로, 전치사/부사는 위치·관계·다른 부사 등으로 통일.

4. **예문/뜻 오타**  
   예문에서 잘못된 뜻 표기( "~를", "~을" 등 오타) 수정.
